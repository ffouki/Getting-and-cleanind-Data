
This code book summarizes the resulting data fields in tidy.txt.

Here are the data we used for the project :

https://d396qusza40orc.cloudfront.net/getdata%2Fprojectfiles%2FUCI%20HAR%20Dataset.zip

Below you can find all variables in the Final_tidy.txt file which results from the run_analysis.R script.

•	activity – The list of all activities and their codes when the measurements were taken 

WALKING (code 1): walking during the test
WALKING_UPSTAIRS (code 2): walking up a staircase during the test
WALKING_DOWNSTAIRS (code 3): walking down a staircase during the test
SITTING (code 4): sitting during the test
STANDING (code 5): standing during the test
LAYING (code 6): laying down during the test

•	subject - The ID of the test and train subject after cbind them

•	features <- The features selected for this database come from the accelerometer and gyroscope 3-axial raw signals tAcc-XYZ and tGyro-XYZ
